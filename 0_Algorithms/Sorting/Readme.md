## <b>Selection Sort</b>

In selection sort, 
1) Find the minimum element in the given array 
2) swap the minimum element with the first element
3) Repeat the above steps for remaining array

Time complexity :
1) Best case : O(n<sup>2</sup>) 
2) Worst case : O(n<sup>2</sup>)
